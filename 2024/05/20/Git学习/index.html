Git学习

设置用户签名

git config –global user.name 用户名

git config –global user.email 邮箱

第一次使用时设置，否则提交代码时会报错

初始化本地库

git init

查看本地库状态

git status

添加文件至暂存区

git add 文件名(eg: hello.txt)

git add * : 提交全部文件至暂存区

将暂存区文件添加到本地库

git commit -m “提交信息” 文件名

查看提交日志

git reflog

git log

将本地仓库代码推送至远端仓库

git push origin master

版本穿梭

git reset –hard 版本号

Git分支操作

创建分支

git branch 分支名

查看分支

git branch -v

切换分支

git checkout 分支名

将指定的分支合并到当前分支

git merge 分支名

合并冲突

1.手动修改冲突位置

2.添加至暂存区

3.提交到本地库（注意：此时git commit -m “信息” 后不能加任何文件名）

查看别名

git remote -v

创建别名

git remote add 别名 http链接（git仓库链接）

建议： 别名与项目名称一致
